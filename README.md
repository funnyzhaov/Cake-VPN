# Cake-VPN
中文版说明

更新服务器列表的说明:
用assets/ directory .ovpn文件替换/添加你的.ovpn文件
现在转到MainActivity.class，找到"getServerList()"方法，你必须更新服务器信息。
[!可选]最后进入SharedPreference.class，找到“getServer()”方法，在那里更新默认的服务器信息。
